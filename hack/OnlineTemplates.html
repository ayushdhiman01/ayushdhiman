JAR - A free Template Provider

For this project i used HTML,CSS,JavaScript,PHP,SQL

What is this project ?

The main objective is that any person with no technical knowledge whatsoever about website development will be able to choose his/her own website from a huge variety of templates and will be able to edit it to his/ her own liking and preference. A new user will be able to sign up and have access to various categories of website templates like - CV, Portfolio, Photography etc. divided into specific sections with easy navigation. The user will be able to choose any template of a website and download the entire code of the website as a zip-file.

Backend

For the back-end i used pHpMyAdmin which stores the login details for both the user and admin. THe webpages accesses this database to fetch and verifies if the credentails are validly entered or not. One more table stores the website pages that the user has saved from his/her account for future use.
